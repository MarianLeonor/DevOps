# Coffee House E-commerce

Proyecto realizado con React y Firebase.

## Tecnologías

- React
- React Router DOM
- Firebase Firestore
- Bootstrap

## Funcionalidades

- Listado de productos
- Detalle de producto
- Categorías
- Carrito
- Checkout
- Generación de órdenes

## Instalación

```bash
npm install
npm run dev
```
